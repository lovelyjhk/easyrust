
## 설치(install)

1. Cargo.toml 파일에 EasyRust를 의존성으로 추가합니다:

    아래 내용을 toml 파일에 추가합니다.

    ```toml
    [dependencies]
    easyrust = "0.1.0"
    ```

2.  다음과 같이 use 라는 키워드로 easyrust 를 호출하세요.

    ```rust
    // main.rs 파일 또는 해당하는 파일에서
    use easyrust;
    ```

3. 메인 함수나 필요한 곳에서 EasyRust 의 함수를 호출해줍니다. 

    ```rust
    fn main() {
        // EasyRust의 함수 호출
    }
    ```
